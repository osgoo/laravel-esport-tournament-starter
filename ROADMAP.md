# Roadmap

## Near Term

- Better bracket formats beyond the two-team demo match
- Tournament check-in flow
- Public tournament pages
- Admin dashboard filters
- More feature and browser tests

## Later

- Double elimination demo mode
- Team invite emails
- Match dispute notes without money/payout workflows
- Optional realtime score updates with documented authorization

## Out of Scope

- Real-money wallets
- Payouts or prize finance
- Payment provider integrations
- Subscription billing
- Tenant rent or SaaS monetization workflows
